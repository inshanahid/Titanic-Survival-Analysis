# 🚢 Titanic Survival Analysis

## 📌 Project Overview
This project analyzes the Titanic dataset to identify factors affecting passenger survival.

## 🎯 Objectives
- Perform data cleaning
- Conduct exploratory data analysis (EDA)
- Visualize survival patterns

## 🛠 Tools & Technologies
- Python
- Pandas
- Seaborn
- Matplotlib

## 📊 Key Insights
- Females had higher survival rates than males
- 1st class passengers had better survival chances
- Age influenced survival (children had higher survival rate)
